# List of endpoints

* '/' - You see a list of endpoints
* '/allBooks' - You get a list of all books
* '/id/:id' - Find a book by it's id.
* '/textreviews' -List the 20 books with most textreviews
* '/author/:author' - Find author
* '/books' - query on author &/or title
